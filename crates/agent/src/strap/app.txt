### os — App Lifecycle (resource: app)
USE THIS when: user wants to launch, quit, list, or manage running applications.

- os(resource: "app", action: "list") — List all running applications
- os(resource: "app", action: "launch", app: "Safari") — Launch an application
- os(resource: "app", action: "quit", app: "Safari") — Quit an application
- os(resource: "app", action: "quit_all") — Quit all applications (except system apps)
- os(resource: "app", action: "activate", app: "Safari") — Bring app to front / focus
- os(resource: "app", action: "hide", app: "Safari") — Hide an application
- os(resource: "app", action: "info", app: "Safari") — Get application details (path, version, etc.)
- os(resource: "app", action: "frontmost") — Get the currently focused application
