### os — System Settings (resource: settings)
USE THIS when: user wants to control volume, brightness, Wi-Fi, Bluetooth, dark mode, or system power.
The action IS the setting name. Value determines the operation (get vs set).

- os(resource: "settings", action: "volume") — Get current volume
- os(resource: "settings", action: "volume", value: 50) — Set volume (0-100)
- os(resource: "settings", action: "mute") — Mute audio
- os(resource: "settings", action: "unmute") — Unmute audio
- os(resource: "settings", action: "brightness") — Get current brightness
- os(resource: "settings", action: "brightness", value: 70) — Set brightness (0-100)
- os(resource: "settings", action: "wifi") — Get Wi-Fi status
- os(resource: "settings", action: "wifi", value: true) — Toggle Wi-Fi
- os(resource: "settings", action: "bluetooth") — Get Bluetooth status
- os(resource: "settings", action: "bluetooth", value: true) — Toggle Bluetooth
- os(resource: "settings", action: "darkmode") — Check dark mode status
- os(resource: "settings", action: "darkmode", value: true) — Toggle dark mode
- os(resource: "settings", action: "battery") — Battery level and charging state
- os(resource: "settings", action: "sleep") — Put system to sleep
- os(resource: "settings", action: "lock") — Lock screen
- os(resource: "settings", action: "info") — System information (hostname, OS, memory, etc.)
