### mcp — Connected MCP Servers

External MCP servers connected through Connectors expose their tools to you directly.

Two things to know:

1. ENUMERATE servers: mcp(action: "list") — lists every connected server and how many
   tools each provides. Use this to answer "what MCP servers/integrations do I have?"

2. CALL a tool: each server's tools appear as their own tools named
   `mcp__<server>__<tool>` (e.g. mcp__monument_sh__project, mcp__neboloop__customer).
   Call them directly by that full name — their arguments are exactly the server's own
   schema (so a tool that needs `resource`/`action` takes them as normal arguments).

Discovering tools you don't see yet: connected MCP tools are activated on demand. Run
tool_search(query: "<server name or capability>") to find the exact proxy name and its
argument schema, then call mcp__<server>__<tool> with the right arguments.

These are live tools from external servers, not skills — call them directly.
