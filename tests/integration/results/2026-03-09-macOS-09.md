## Publisher's Guide Discrepancies Found

1. **Workflow `fallback`** — Guide describes fallback as arbitrary activity reference. Implementation uses enum: `notify_owner`, `skip`, `abort`.
2. **Activity `tools` field** — Guide says "tools restriction". Implementation uses `skills` field (skill names, not tool names).
3. **Role triggers** — Guide describes role.json with triggers parsed and registered. Implementation: `role(create)` only creates ROLE.md + manifest.json, no role.json. `role(info)` doesn't show trigger details.
4. **Skill triggers** — Guide mentions triggers in frontmatter. Catalog shows capabilities but not triggers.

## Blocking Issues

1. **Marketplace install (S-10, W-09, R-07)** — Returns 404 "invalid code". Blocks sealed .napp, namespace, qualified names, and dependency cascade tests.
2. **Sub-agent spawn (AT-17)** — "No messages in session" error. Sub-agents can't execute.
3. **Role trigger parsing** — role.json not auto-generated or parsed by role tool. Triggers only in ROLE.md text.
